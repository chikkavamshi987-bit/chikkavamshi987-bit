# 📚 BookWorm 🐛

BookWorm is a community-driven reading platform built for book lovers. Users can share their favorite books, write reviews, rate reads, and discover what others are reading. Whether it's fiction, non-fiction, or self-help, BookWorm helps readers find their next great book.

## 🚀 App Features

🔐 Authentication  
- Secure signup & login using JWT  
- Proper error handling for invalid credentials  

🏠 Home Feed  
- Newest-first posts  
- Infinite scrolling  

➕ Create Post  
- Add title, rating, cover image, and caption  

👤 Profile Screen  
- User profile information  
- Displays all posts created by the user  

🗑️ Delete Post  
- Confirmation alert before removing posts  

🎨 Themes  
- 4 instant UI themes using a single color object  

🌐 Web Support  
- Runs on localhost in browser  

🚪 Logout functionality

# 💰 Budget Tracker App

A modern **React Native mobile application** that helps users track income, expenses, and financial summaries efficiently.

## 🚀 Features

- 🔐 User Authentication (Register, Login, Email OTP Verification)  
- 📊 Expense Summary Dashboard  
- ➕ Add & Delete Transactions  
- 📄 Paginated Transaction History  
- 📡 Network Connectivity Detection (Online / Offline Mode)  
- 🔒 Secure backend with JWT authentication  

## 📱 App Functionality

- User registration with email OTP verification
- Secure login using JWT
- View income, expenses, and balance summary
- Add and delete financial transactions
- Pagination support for transaction history
- Offline network detection using NetInfo

## 🔐 Security

- Password encryption using bcrypt
- JWT token-based authentication
- Email verification with OTP expiry
